## Giới thiệu

Tài liệu này hướng dẫn cách tạo và quản lý icon trong dự án bằng cách sử dụng IcoMoon để chuyển đổi file SVG thành font icon, và cách generate các type TypeScript tương ứng để sử dụng trong dự án.

## Import SVG vào IcoMoon

IcoMoon là một công cụ mạnh mẽ cho phép bạn tạo và quản lý font icon từ các file SVG. Dưới đây là các bước để import SVG vào IcoMoon:

1. **Chuẩn bị file SVG**:

   - Đảm bảo file SVG của bạn có định dạng phù hợp, tốt nhất là đơn sắc và tối ưu hóa kích thước.
   - Đặt tên file theo quy tắc rõ ràng, ví dụ: `icon-name.svg`.

2. **Truy cập IcoMoon App**:

   - Mở trình duyệt và truy cập [IcoMoon App](https://icomoon.io/app/#/select).

3. **Tạo Project mới**:

   - Nhấn vào menu "Manage Projects" (biểu tượng bánh răng).
   - Chọn "New Project" để tạo dự án mới hoặc import dự án hiện có.

4. **Import SVG**:

   - Trong giao diện chính của IcoMoon, nhấn nút "Import Icons" (hoặc kéo thả các file).
   - Chọn các file SVG từ máy tính của bạn.
   - Các icon sẽ xuất hiện trong danh sách.

5. **Chỉnh sửa và cấu hình Icon**:

   - Bạn có thể điều chỉnh kích thước, căn chỉnh và chỉnh sửa từng icon.
   - Đảm bảo mỗi icon có tên duy nhất và dễ nhớ.

6. **Chọn Icon để tạo Font**:

   - Chọn tất cả các icon bạn muốn đưa vào font (bằng cách nhấp vào từng icon).
   - Nhấn nút "Generate Font" ở cuối trang.

7. **Cấu hình Font**:

   - Đặt tên cho font và cấu hình các tùy chọn như tiền tố, định dạng, giá trị Unicode.
   - Kiểm tra xem preview hiển thị đúng không.

8. **Tải xuống Font**:
   - Nhấn nút "Download" để tải xuống font icon đã tạo.
   - File tải về sẽ chứa các font file (.woff, .ttf, ...), CSS và HTML demo.

## Generate Type Icon

Để sử dụng các icon này với TypeScript, chúng ta cần tạo các type tương ứng:

1. **Tạo file Type cho Icon**:

   - Sau khi tải font xuống từ IcoMoon, đặt file `selection.json` vào thư mục `src/assets/icons/` (nếu chưa có, hãy tạo thư mục này).

2. **Chạy script để generate type**:

   - Dự án đã được cấu hình sẵn script để generate các type từ file selection.json
   - Chạy lệnh sau để tạo type cho icon:

   ```bash
   yarn generate:icon-types
   ```

   - Script sẽ tự động đọc file selection.json và tạo ra file type tương ứng tại `src/types/IconType.ts`

3. **Kiểm tra file type đã được tạo**:

   - Mở file `src/types/IconType.ts` để kiểm tra xem tất cả các icon đã được thêm vào type IconName chưa
   - File type có dạng như sau:

   ```typescript
   // Auto-generated by generateIconTypes script
   // DO NOT EDIT MANUALLY

   export type IconName =
     | "home"
     | "search"
     | "user"
     // ... các icon khác
     | "settings";
   ```

## Sử dụng Icon trong dự án

Sau khi đã tạo font và type cho icon, bạn có thể sử dụng chúng trong dự án như sau:

1. **Sử dụng component Icon**:

   ```tsx
   import Icon from "../components/Icon";
   ```

Với cách tiếp cận này, bạn có thể:

- Quản lý tất cả icon trong một font duy nhất
- Có type checking cho tên icon
- Thêm/xóa icon dễ dàng bằng cách cập nhật font và chạy lại script generate
- Đảm bảo tính nhất quán trong toàn bộ ứng dụng
